# Store Inventory Suite Web

React + Vite frontend for the Store Inventory Suite.

## Prerequisites

- Node.js 20+

## Setup

```bash
npm install
cp .env.example .env
npm start
```

- App URL: http://localhost:5173

## Environment Variables

See [.env.example](.env.example).

- VITE_API_BASE_URL

## Scripts

- npm start
- npm run build
- npm run lint
